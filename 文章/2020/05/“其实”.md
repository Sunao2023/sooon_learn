```text
“其实”
```

## 问题

```text
当人们说“其实”时，他们为什么要用这个词？
```

## 回答

```text
为了在肯定的基础之上否定。

用“其实”“实际上”，表示不否定前述的观察，但要做更准确的概括。

这个“更准确的概括”，从另一个角度看毕竟也同时是一个“不同的概括”。

可以说是一种特殊的否定。
```
